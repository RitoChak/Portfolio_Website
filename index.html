Google Data Analytics Capstone: Cyclistic Case Study

Conducted comprehensive data cleaning and analysis of Cyclistic's 2023 trip data using SQL and created insightful visualizations in Tableau. This project contains the exploratory data analysis and model training of a student performance indicator dataset. The data cleaning, analysis and visualization has been done on Jupyter Notebook. The visualization has also been done on tableau.
