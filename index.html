Machine Learning Specialization

Certificate of Completion


The Machine Learning Specialization is a foundational online program created in collaboration between DeepLearning.AI and Stanford Online. It provides a broad introduction to modern machine learning, including supervised learning (multiple linear regression, logistic regression, neural networks, and decision trees), unsupervised learning (clustering, dimensionality reduction, recommender systems), and some of the best practices used in Silicon Valley for artificial intelligence and machine learning innovation (evaluating and tuning models, taking a data-centric approach to improving performance, and more.)

Deep Learning Specialization

Certificate of Completion


The Deep Learning Specialization is a foundational program that helps in understanding the capabilities, challenges, and consequences of deep learning and prepares to participate in the development of leading-edge AI technology.

Mathematics for Machine Learning and Data Science Specialization

Certificate of Completion


Mathematics for Machine Learning and Data Science is a foundational online program created by DeepLearning.AI . This Specialization is to master the fundamental mathematics toolkit of machine learning. It consists of 3 courses : Linear Algebra, Calculus and Probablity & Statistics.
